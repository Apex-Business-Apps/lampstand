# LampStand — Mission Statement

LampStand is a non-profit scripture companion application built as a gift to the community.

## Core Principles

- **No monetization.** LampStand does not charge for access, does not display ads, and does not sell user data.
- **Privacy-first.** All scripture study data remains local to the user's device. No cloud sync of personal study notes without explicit opt-in.
- **Accessibility.** LampStand must remain free and accessible to every user regardless of income or location.
- **Mission-locked.** This project will never be converted to a commercial product.

## What This Means for Contributors

- Do not add Stripe, in-app purchase, or subscription logic.
- Do not add advertising SDKs or analytics that monetize user behavior.
- Do not add paywalls or feature gates tied to payment.
- Do not add telemetry that tracks personally identifiable information.

If you are unsure whether a feature aligns with the mission, ask before building.
